# Flight Prices Monitoring Pipeline

## Overview
This project is a data engineering pipeline that collects flight price data, stores historical records, and enables price change detection over time.

## Objectives
- Practice real-world data ingestion and storage
- Build reliable ETL workflows
- Track and analyze flight price trends
- Prepare a production-style data pipeline

## Tech Stack (Planned)
- Python
- PostgreSQL
- Airflow
- Docker
